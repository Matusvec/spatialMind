    1. Create server/services/query_walker.py with class QueryWalker(WalkerBase):
       - __init__(self, scene_graph: dict, query: str, memory_service=None, scene_id: str = "default"):
         - super().__init__(scene_graph)
         - self.query = query.lower()
         - self.memory_service = memory_service
         - self.scene_id = scene_id
         - self.matched_nodes: list[dict] = []
         - self.spatial_context_parts: list[str] = []

       - is_relevant(self, node: dict) -> bool:
         - Simple keyword matching: check if any word in self.query appears in node["label"].lower()
         - Also check if query appears as substring of label
         - Return True if relevant, False otherwise
         - NOTE: In a full implementation, this would use CLIP similarity. For hackathon, keyword matching on node labels is sufficient since labels come from CLIP-clustered objects.

       - on_node(self, node: dict, depth: int) -> bool:
         - If self.is_relevant(node):
           - Append node to self.matched_nodes
           - Find edges connected to this node
           - Build context line: "{label} ({id}) at ({cx:.1f}, {cy:.1f}, {cz:.1f})"
           - For each edge, add relationship: "  - {relation} {neighbor_label} (confidence: {conf:.2f})"
           - Append context lines to self.spatial_context_parts
         - Return True (continue traversal — we want to find ALL relevant nodes)

       - build_spatial_context(self) -> str:
         - If no matched_nodes: return "No objects matching the query were found in the scene."
         - Join spatial_context_parts with newlines
         - Prepend: "Found {N} objects matching '{self.query}':\n"
         - Return the full context string

       - async run(self) -> dict:
         - Call self.traverse_all() to visit all nodes
         - Build spatial_context = self.build_spatial_context()
         - Collect all gaussian_indices from matched_nodes (flat list)
         - Collect matched node summaries for response

         - If self.memory_service is not None:
           - answer = await self.memory_service.query_with_context(self.scene_id, self.query, spatial_context)
         - Else:
           - answer = spatial_context  # Fallback: return raw spatial facts as answer

         - Return:
           ```python
           {
               "answer": answer,
               "query": self.query,
               "matched_nodes": [{"id": n["id"], "label": n["label"], "centroid": n["centroid"]} for n in self.matched_nodes],
               "highlight_indices": gaussian_indices,  # flat list for frontend highlight mask
               "scene_id": self.scene_id,
           }
           ```
