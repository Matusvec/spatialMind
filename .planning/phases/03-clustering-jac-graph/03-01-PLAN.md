  <action>
Create `server/services/clustering.py` with two public functions:

1. `compute_dynamic_epsilon(positions: np.ndarray, min_samples: int = 6) -> float`:
   - Compute k-distance graph: for each point, find distance to its k-th nearest neighbor using `scipy.spatial.KDTree`
   - Sort distances ascending
   - Find elbow using maximum curvature: compute second derivative of sorted distances, pick index of max absolute second derivative
   - Return the k-distance at the elbow index
   - Fallback: if elbow detection fails (flat curve), return median k-distance
   - Per PITFALLS.md Pitfall 6: NEVER hardcode epsilon

2. `cluster_gaussians(positions: np.ndarray, embeddings: np.ndarray, scores: np.ndarray, indices: np.ndarray, min_samples: int = 6) -> list[dict]`:
   - `positions`: (M, 3) float32 — XYZ of the M selected Gaussians
   - `embeddings`: (M, 512) float32 — CLIP embeddings of selected Gaussians
   - `scores`: (M,) float32 — similarity scores from top_k
   - `indices`: (M,) int — original Gaussian indices (into the full store)
   - Compute dynamic epsilon via `compute_dynamic_epsilon(positions, min_samples)`
   - Run `sklearn.cluster.DBSCAN(eps=epsilon, min_samples=min_samples, algorithm='ball_tree')` on positions
   - For each unique cluster label (excluding -1 noise):
     - `label`: int cluster ID
     - `centroid`: mean of cluster positions as [x, y, z] list
     - `bbox`: [x_min, y_min, z_min, x_max, y_max, z_max] from min/max of cluster positions
     - `gaussian_indices`: original indices (from `indices` param) for this cluster's members, as int list
     - `confidence`: mean similarity score of cluster members (from `scores`)
     - `mean_embedding`: mean of cluster member embeddings, L2-normalized, as (512,) ndarray
   - Return list of cluster dicts sorted by confidence descending
   - Log: number of clusters found, number of noise points, epsilon used

  <action>
Create `server/services/spatial_relations.py` with two public functions:

1. `extract_spatial_relations(clusters: list[dict]) -> list[dict]`:
   - For each pair of clusters (i, j) where i != j:
   - **on_top_of check**: Compare Y-axis (centroid[1]) — if cluster_i.centroid[1] - cluster_j.centroid[1] > y_threshold AND their XZ distance is within proximity:
     - y_threshold = 0.3 * scene_scale (scene_scale = max extent of all cluster centroids)
     - XZ distance = sqrt((cx_i - cx_j)^2 + (cz_i - cz_j)^2)
     - XZ proximity threshold = 0.5 * scene_scale
     - Confidence: 1.0 - (overlap_ratio between bboxes in Y-axis). If no overlap, confidence = 1.0. If partial overlap, confidence = 1.0 - overlap_fraction. Min confidence 0.1.
     - Edge: `{"from_label": i, "to_label": j, "relation": "on_top_of", "confidence": float, "distance": float}`
   - **next_to check**: XZ plane distance between centroids:
     - If XZ_distance < proximity_threshold AND abs(Y-offset) < y_threshold (roughly same level):
     - Confidence: 1.0 - (XZ_distance / proximity_threshold). Closer = higher confidence.
     - Edge: `{"from_label": i, "to_label": j, "relation": "next_to", "confidence": float, "distance": float}`
   - Deduplicate: for next_to, only emit edge for (i, j) where i < j (symmetric). For on_top_of, emit directed edge (higher on top of lower).
   - Return list of edge dicts sorted by confidence descending.
   - Use scene-relative thresholds (compute scene_scale from all centroids) so it works for any coordinate space — per PITFALLS.md Pitfall 6.

2. `build_hierarchy(clusters: list[dict], similarity_threshold: float = 0.7) -> list[dict]`:
   - For each pair of clusters, compute cosine similarity between their `mean_embedding` vectors
   - If similarity > similarity_threshold, emit a hierarchy edge:
     - `{"from_label": i, "to_label": j, "relation": "similar_to", "similarity": float}`
   - This creates the emergent hierarchy from CLIP embedding proximity per GRAPH-02
   - Return list of hierarchy edge dicts sorted by similarity descending
