**Section 3: Scaniverse Capture Guide (per D-01, D-02, D-03)**
- Use iPhone with Scaniverse app
- Select "Photo Mode" (not LiDAR scan mode — we need raw photos for COLMAP)
- Walk slowly around the target area, capturing 50-80 photos
- Overlap between photos: ~60-70% (needed for COLMAP feature matching)
- Include objects at multiple heights: floor items, table items, wall items
- Avoid: featureless walls, reflective surfaces, moving people
- Good subjects: tables with items on them, chairs, monitors, backpacks, water bottles
- Export: Share -> Save Photos -> Upload to Google Drive `/spatialMind/data/jachacks_venue/input/`

**Section 4: Scene Switching in Notebook**

**Section 5: Troubleshooting**
- **Colab disconnects mid-training:** Reconnect, run Cell 0 (remount Drive), check for existing checkpoints in Cell 6, re-run from where it stopped
- **COLMAP fails (no cameras.bin):** Check that photos are in `{DATASET_PATH}/input/` directory, verify photos have EXIF data, try with `--ImageReader.single_camera 1` flag
- **Smoke test fails (std < 0.05):** Embeddings are noise. Check: (1) was RGB checkpoint used for LangSplat training? (2) did autoencoder train on correct language_features directory? (3) try increasing 3DGS iterations to 40K
- **SAM preprocessing fails:** Verify SAM checkpoint path matches, check GPU memory (SAM vit_h needs ~8 GB VRAM)
- **"CUDA error: no kernel image":** PyTorch CUDA version mismatch. Re-run Cell 1 with the exact `torch==2.0.1+cu118` command
