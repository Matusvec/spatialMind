    1. Create server/services/exploration_walker.py with class ExplorationWalker(WalkerBase):
       - __init__(self, scene_graph: dict, memory_service=None, scene_id: str = "default"):
         - super().__init__(scene_graph)
         - self.memory_service = memory_service
         - self.scene_id = scene_id
         - self.catalog_entries: list[dict] = []

       - on_node(self, node: dict, depth: int) -> bool:
         - Collect node info: id, label, centroid, confidence
         - Find edges connected to this node (from self.edges)
         - Build relationships list: ["next_to obj_002 (conf: 0.75)", "on_top_of obj_003 (conf: 0.90)"]
         - Append to self.catalog_entries: {"id": node["id"], "label": node["label"], "centroid": node["centroid"], "confidence": node["confidence"], "relationships": relationships}
         - Always return True (explore everything, never disengage early)

       - format_catalog(self) -> str:
         - Build natural language catalog from self.catalog_entries
         - Format:
           ```
           Scene Object Catalog ({N} objects):

           - object_0 (obj_000): located at (1.2, 0.5, -0.8), confidence 0.85
             Relationships: next_to object_1, on_top_of object_2
           - object_1 (obj_001): located at (2.0, 0.3, 1.1), confidence 0.72
             Relationships: next_to object_0
           ```
         - For each entry, format centroid as (x, y, z) with 1 decimal
         - List relationships with target node labels

       - async run(self) -> dict:
         - Call self.traverse_all() to visit all nodes
         - Generate catalog via self.format_catalog()
         - If self.memory_service is not None, call await self.memory_service.store_catalog(self.scene_id, catalog)
         - Return {"catalog": catalog, "object_count": len(self.catalog_entries), "scene_id": self.scene_id}
