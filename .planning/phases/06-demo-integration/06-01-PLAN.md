Create `scripts/demo_launch.py` — a Python script that serves as the single entry point for the hackathon demo. Use standard library only (subprocess, time, urllib, json, sys, os). No external dependencies.

The script must:

1. **Print a banner**: "SpatialMind Demo Launcher" with the three demo beats listed.

2. **Check scene availability** (before starting server):
   - Check if `./data/scene/artifacts/point_cloud.ply` exists (live scene).
   - If not, check if `./data/fallback_scene/artifacts/point_cloud.ply` exists (fallback).
   - If neither exists, print a clear error: "No scene data found. Place PLY artifacts in ./data/scene/artifacts/ or ./data/fallback_scene/artifacts/" and exit(1).
   - Print which scene will be used: "[LIVE] Using JacHacks venue scene" or "[FALLBACK] Using pre-trained scene".

3. **Start the backend server**:
   - Launch `python -m uvicorn server.main:app --host 0.0.0.0 --port 8001` as a subprocess (not blocking).
   - Poll `http://localhost:8001/api/health` every 2 seconds, up to 60 seconds total.
   - Wait until the response contains `"pipeline_ready": true`.
   - Print progress: "Waiting for server..." then "Server ready! Pipeline: OK" or timeout error.

4. **Print demo instructions** once server is ready:
   ```
   === DEMO READY ===

   Frontend: http://localhost:5173
   Backend:  http://localhost:8001
   Health:   http://localhost:8001/api/health

   Three-Beat Demo Sequence:
     Beat 1: Click "Begin Exploration", then type "chairs" in chat
             -> Chair Gaussians light up in <5s
     Beat 2: Type "what's on the tables?"
             -> Natural language answer with real objects
     Beat 3: Type any unrehearsed query (e.g., "where are the windows?")
             -> Spatially correct response

   Press Ctrl+C to stop.
   ```

5. **Wait for Ctrl+C**, then kill the server subprocess cleanly.

Make the script executable with `if __name__ == "__main__":` guard. Use `argparse` for an optional `--port` flag (default 8001) and `--scene-dir` flag (default `./data/scene`).
