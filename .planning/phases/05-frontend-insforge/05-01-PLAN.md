Create server/routers/adapter.py with an APIRouter(prefix="/api") containing these endpoints:

1. **GET /api/health** — Proxy to internal health endpoint. Return {"status": "ok", ...passthrough}. The frontend checks `data.status === "ok"` at line 86 of Index.tsx.

2. **POST /api/explore/start** — Trigger scene build (POST /scene/build with defaults) then exploration (POST /walker/explore). Store result in app_state["adapter_explore_result"]. Return {"status": "started"}. The frontend calls this at line 180 and expects a simple JSON ack.

3. **GET /api/explore/status** — Return the cached explore result shaped for the frontend. The frontend polls this at line 185-289 expecting:
   - `status`: "exploring" or "complete"
   - `objects`: array of {id, label, confidence, position, category} — derive from scene_graph nodes
   - `graph`: {nodes: [{id, label, type}], edges: [{from, to, relation}]} — derive from scene_graph
   - `progress`: float 0-1
   If explore is not yet complete, return status="exploring" with partial data from scene_graph nodes (which are available after /scene/build).
   If complete, return status="complete" with full objects from exploration catalog + scene graph.

   Map scene graph nodes to frontend DetectedObject format:
   - id: node.id (e.g., "obj_001")
   - label: node.label (e.g., "Chair")
   - confidence: node.confidence
   - position: node.centroid (3-element list)
   - category: derive from label using simple heuristic ("Furniture" for chair/table/shelf, "Electronics" for monitor/laptop, "Lighting" for lamp/light, "Decor" for plant/rug, "Structure" for window/wall/door, default "Objects")

   Map scene graph edges to frontend GraphEdge format:
   - from: edge.from_id
   - to: edge.to_id
   - relation: edge.relation

4. **POST /api/query** — Accept {"query": str, "session_id": str}. Call internal walker/query with {"query": query, "scene_id": session_id or "default"}. Shape response:
   - `description`: walker response "answer" field (frontend checks data.description || data.answer at line 347)

   - `nodes`: walker response "matched_nodes" mapped to [{label, confidence: 0.9}] (frontend reads data.nodes at line 357)

For the category heuristic, create a simple function:
```python
CATEGORY_MAP = {
    "chair": "Furniture", "table": "Furniture", "desk": "Furniture",
    "shelf": "Furniture", "couch": "Furniture", "sofa": "Furniture",
    "bookshelf": "Furniture", "cabinet": "Furniture", "armchair": "Furniture",
    "monitor": "Electronics", "laptop": "Electronics", "screen": "Electronics",
    "lamp": "Lighting", "light": "Lighting",
    "plant": "Decor", "rug": "Decor", "vase": "Decor", "painting": "Decor",
    "window": "Structure", "wall": "Structure", "door": "Structure", "floor": "Structure",
}

def categorize(label: str) -> str:
    lower = label.lower()
    for keyword, category in CATEGORY_MAP.items():
        if keyword in lower:
            return category
    return "Objects"
```
