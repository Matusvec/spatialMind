Build the graph builder service and scene build endpoint that orchestrates DBSCAN clustering, spatial relationship extraction, and hierarchy computation into a single scene graph payload.

Purpose: This is the public API that triggers scene graph construction. The endpoint is called once at scene load (not per-query) and produces the full node+edge payload that will be consumed by the JAC graph in Phase 4.   <action>
Create `server/services/graph_builder.py` with one public function:

`build_scene_graph(gaussian_store: GaussianStore, clip_encoder, query: str = "objects", k: int = 5000, min_samples: int = 6, hierarchy_threshold: float = 0.7) -> dict`:

This is the orchestration function. Steps:
1. Encode `query` via `clip_encoder.encode_text(query)` to get a broad text embedding (default "objects" captures everything)
2. Run `top_k(text_embedding, gaussian_store.decoded_embeddings, k=k)` to get the top-K Gaussian indices and scores
3. Extract positions for selected Gaussians: `gaussian_store.positions[indices]` -> (M, 3)
4. Extract embeddings for selected Gaussians: `gaussian_store.decoded_embeddings[indices]` -> (M, 512)
5. Call `cluster_gaussians(selected_positions, selected_embeddings, scores, indices, min_samples)` -> cluster list
6. Call `extract_spatial_relations(clusters)` -> edge list
7. Call `build_hierarchy(clusters, hierarchy_threshold)` -> hierarchy list
8. Build response dict:

```python
{
    "nodes": [
        {
            "id": f"obj_{i:03d}",
            "label": f"object_{i}",  # placeholder — Phase 4 adds LLM descriptions
            "centroid": cluster["centroid"],
            "bbox": cluster["bbox"],
            "gaussian_indices": cluster["gaussian_indices"],
            "confidence": cluster["confidence"],
        }
        for i, cluster in enumerate(clusters)
    ],
    "edges": [
        {
            "from": f"obj_{edge['from_label']:03d}",
            "to": f"obj_{edge['to_label']:03d}",
            "relation": edge["relation"],
            "confidence": edge["confidence"],
            "distance": edge["distance"],
        }
        for edge in spatial_edges
    ],
    "hierarchy": [
        {
            "from": f"obj_{h['from_label']:03d}",
            "to": f"obj_{h['to_label']:03d}",

            "similarity": h["similarity"],
        }
        for h in hierarchy_edges
    ],
    "metadata": {
        "gaussian_count": gaussian_store.count,
        "selected_count": len(indices),
        "cluster_count": len(clusters),
        "edge_count": len(spatial_edges),
        "hierarchy_count": len(hierarchy_edges),
        "query": query,
        "k": k,
    }
}
```

Define Pydantic models:
- `SceneBuildRequest(BaseModel)`: query (str, default "objects"), k (int, default 5000), min_samples (int, default 6), hierarchy_threshold (float, default 0.7), force (bool, default False)
- `SceneNodeResponse(BaseModel)`: id (str), label (str), centroid (list[float]), bbox (list[float]), gaussian_indices (list[int]), confidence (float)
- `SceneEdgeResponse(BaseModel)`: from_id (str, alias="from"), to_id (str, alias="to"), relation (str), confidence (float), distance (float)
- `SceneHierarchyResponse(BaseModel)`: from_id (str, alias="from"), to_id (str, alias="to"), relation (str), similarity (float)
- `SceneMetadataResponse(BaseModel)`: gaussian_count (int), selected_count (int), cluster_count (int), edge_count (int), hierarchy_count (int), query (str), k (int)
- `SceneBuildResponse(BaseModel)`: nodes (list[SceneNodeResponse]), edges (list[SceneEdgeResponse]), hierarchy (list[SceneHierarchyResponse]), metadata (SceneMetadataResponse)

Endpoint `POST /scene/build`:
- Import get_app_state from server.main
- Guard: 503 if gaussian_store not loaded or clip_encoder not initialized (same pattern as clip.py)

- Call `build_scene_graph(gaussian_store, clip_encoder, query, k, min_samples, hierarchy_threshold)`
- Store result in app_state["scene_graph"]
- Return SceneBuildResponse
