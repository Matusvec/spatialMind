    1. Create server/services/walker_base.py with class WalkerBase:
       - __init__(self, scene_graph: dict, max_depth: int = 100):
         - self.nodes = {n["id"]: n for n in scene_graph["nodes"]}
         - self.edges = scene_graph["edges"]
         - self.adjacency: dict[str, list[str]] = build from edges (both from->to and to->from for undirected traversal)
         - self.visited: set[str] = set()
         - self.max_depth = max_depth
         - self.results: list = []
       - get_node(self, node_id: str) -> dict | None: Return self.nodes.get(node_id)
       - get_neighbors(self, node_id: str) -> list[str]: Return self.adjacency.get(node_id, [])
       - get_edge(self, from_id: str, to_id: str) -> dict | None: Find edge between the two nodes
       - on_node(self, node: dict, depth: int) -> bool: Override hook. Called when visiting a node. Return True to continue traversal, False to disengage. Default returns True.
       - traverse(self, start_id: str | None = None) -> list: BFS traversal from start_id (or first node if None). Uses self.visited set to skip already-visited nodes. Respects max_depth. Calls self.on_node() for each visited node. Returns self.results.
       - traverse_all(self) -> list: Traverse from every unvisited root node (handles disconnected components). Returns self.results.

    2. Build adjacency list from edges:
       ```python
       adjacency: dict[str, list[str]] = {}
       for edge in scene_graph["edges"]:
           src, dst = edge["from"], edge["to"]
           adjacency.setdefault(src, []).append(dst)
           adjacency.setdefault(dst, []).append(src)
       ```

    3. BFS traversal in traverse():
       ```python
       from collections import deque
       queue = deque([(start_id, 0)])  # (node_id, depth)
       while queue:
           node_id, depth = queue.popleft()
           if node_id in self.visited or depth > self.max_depth:
               continue
           self.visited.add(node_id)
           node = self.get_node(node_id)
           if node is None:
               continue
           should_continue = self.on_node(node, depth)
           if not should_continue:
               break  # disengage
           for neighbor_id in self.get_neighbors(node_id):
               if neighbor_id not in self.visited:
                   queue.append((neighbor_id, depth + 1))
       return self.results
       ```
